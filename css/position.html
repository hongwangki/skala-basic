sticky: 스크롤하면 이 바가 상단(top:0)에 붙어서 따라다님
아래를 스크롤하면서 확인해봐.
- static: 기본값, top/left 적용 안 됨
- relative: 원래 자리 기준으로 이동(자리 차지 유지)
- absolute: 가장 가까운 position 조상(.card) 기준으로 배치
- fixed: 화면(뷰포트) 기준으로 고정(오른쪽 아래)
- sticky: 특정 위치(top)부터 화면에 붙음(상단 바)
1) position: static
STATIC
top/left를 줘도 이동하지 않음(무시됨)
2) position: relative
RELATIVE
원래 자리에서 top/left만큼 "보이는 위치"만 이동
원래 자리(공간)는 그대로 차지
3) position: absolute
ABSOLUTE
이 박스는 .card(=position: relative)를 기준으로
top/right에 붙어서 배치됨(문서 흐름에서 빠짐)
fixed / sticky 확인용 스크롤 구간(여길 스크롤해봐)
- 오른쪽 아래에 FIXED 박스가 계속 따라다니는지 확인
- 위의 sticky-bar가 상단에 붙어있는지 확인
4) position: fixed
fixed는 카드 안이 아니라 화면 기준이라서
이 카드와 무관하게 오른쪽 아래에 떠 있음
5) position: sticky
sticky는 스크롤하다가 top: 0 위치가 되면
화면 상단에 "붙어서" 내려가지 않음 (위 보라색 바)
FIXED
